Chapter 1: Information Technology, the Internet and You

Basic Terminologies

  • Data: Individual facts or collection of individual facts or statistics. Raw form that doesn’t carry any significance or purpose.
  • Information: The result of analyzing and interpreting pieces of data, it is the perception of those pieces of knowledge.
  • Connectivity: Is a concept related to using computer networks to link people and resources.
  • Internet: Is a globally connected network system that facilitates communication and data services.
  • Web: Provides multimedia interface to resources available on the internet.

Driving Impact of Technology

  • Cloud Computing: Delivery of computing services including servers, storage, databases, network, etc. – over the internet.
  • Wireless Revolution: Allows individuals to stay connected with one another from almost anywhere at any time.
  • Internet of Things (IoT): Describes the network of physical objects – ‘things’ – that are embedded with sensors, software, and other technologies.

Information System

Information Systems are collections of multiple information resources, to gather, process, store, and disseminate information.

Components of Information System:

  • People
  • Software
  • Hardware
  • Data
  • Information

Basic Computer Operations

  • Input Operation – the computer receives information using input devices like keyboard, mouse, or any other devices.
  • Storage Operation – The given information stored in a computer using the different storage devices.
  • Processing Operation – it is considered the basic computing operation. It executes the instructions, control storage data, etc.
  • Output Operation – represents the results from the operations of CPU.

Primary Memory

  • Random Access Memory (RAM) - Is computer memory that a processor or computer accesses first or directly.
  • Read-Only Memory (ROM) - A type of storage medium that permanently stores data.
  • Cache Memory - Is a small-sized type of volatile computer memory that provides high-speed data access to processor.
  • Register - Is one of a small set of data holding places that are part of the computer processor.

Secondary Memory:

  • Magnetic Tape - Well-suited for archiving because of its high capacity, low cost, and durability.
  • Hard Disk - Most basic of hard drives consist of disk platters that are positioned around a spindle.
  • CD/DVD - Optical disc technology with 4.7gb storage capacity on a single-sided, one-layered disk.
  • Flash Memory - Is a type of non-volatile memory that erases data in units called a block.
